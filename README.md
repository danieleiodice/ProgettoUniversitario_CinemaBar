# ProgettoUniversitario_CinemaBar
Progetto universitario in C++ per gestire un cinema e il bar. Permette di creare, visualizzare e vendere biglietti e prodotti, con tre ruoli principali. Ideato per esercitare la programmazione orientata agli oggetti, la gestione dinamica della memoria e l’interazione con l’utente.

## Funzionalità
- Gestione biglietti dei film (creazione, visualizzazione, rimozione)
- Gestione prodotti del bar (aggiunta, visualizzazione, rimozione, vendita)
- Gestione vendite e incassi tramite lista dinamica
- Tre ruoli principali: Fornitore Biglietti, Fornitore Bar, Addetto alla Cassa

## Strumenti utilizzati
- C++ (standard library)
- File I/O per salvataggio dati
- Programmazione orientata agli oggetti

## Compilazione
```
g++ main.cpp -o cinemaBar
```
## Esecuzione
```
./cinemaBar
```
